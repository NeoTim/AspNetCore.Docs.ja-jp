# <a name="aspnet-core-docs-whats-new-for-june-1-2020---june-30-2020"></a>ASP.NET Core docs: 2020 年6月1日の新機能-2020 年6月30日

## <a name="migration"></a>移行

### <a name="updated-articles"></a>更新された記事
